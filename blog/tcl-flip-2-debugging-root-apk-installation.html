TCL Flip 2 (T408DL) USB Debugging, Root, and apk installation on phones that *#*#DEBUG#*#* is blocked

Published: June 18, 2025

This guide is only for the Tcl Flip 2 (T408DL). Be aware of the firmware you are on. Some parts of the guide applicable to all firmware versions.

I have been out of the loop for a while, and see that some have a problem with new KEKA firmware. Thank you Arisu Sanchez for posting this link in the comments: https://stene.xyz/flip2/

USB Debugging, Root, and APK Installation

On phones where *#*#DEBUG#*#* is blocked

Credits

  • [neutronscott] (GitHub, Apps4flip) – Modified boot image & Magisk mouse module.
  • [u/__Master__Of__None__] (Reddit) – MTKFastboot & resolution tool.
  • [u/chet50] (Reddit) – APK for enabling dev options if rooted.
Important: This guide will fully format your device. Make sure to back up your data first.

Instructions

  1. Download and extract the zip: TCL Flip 2 Files
  2. Open "mtkfastboot & drivers", install mtkdrivers.exe, then reboot your PC.
  3. Reopen folder and run mtkfastboot.exe. A command window should say "com port found…waiting".
  4. Turn off your phone, wait a few seconds, then plug it in. It should enter fastboot. If not, try holding both volume buttons while plugging in.
  5. Open "platform tools, files for flashing, & apks" and open CMD in that folder.

Run these commands:

fastboot devices
fastboot flashing unlock
fastboot flash boot neutron.img
fastboot reboot

After Reboot

  1. Dial *#*#217703#*#*, choose Magisk, and install full version.
  2. Reboot, and repeat the dialer code if needed until Magisk shows version 25.2 installed.
  3. Enable Zygisk in Magisk settings and reboot.

Install ADBIO APK

Open browser on your phone and go to: https://tinyurl.com/adbioapk

  1. Click on the download notification (not File Manager).
  2. Allow unknown sources.
  3. Click Back and install the app.
  4. Use launcher code, open app, grant root, click enable.

ADB Commands

adb devices
adb shell settings put global development_settings_enabled 1
adb install mrepo.apk
adb install reschanger.apk
adb install esfileexplorer.apk
adb install a4faccessibility.apk
Optional: wm size 321x428 – Adjusts resolution for more screen space. Can toggle using reschanger.

Use mrepo to flash v2mouse.zip

Use ES File Explorer to browse and flash. Reboot after flashing.

Mouse Controls

  • Activate: Long press *
  • Deactivate: Press *
  • Left Click: OK or 1
  • Right Click: 3
  • Speed Up/Down: Vol Up/Down
  • Horizontal Scroll: 4 / 6
  • Vertical Scroll: 7 / * or 9 / #

Done!

Congrats! Enjoy your liberated Flip 2!
